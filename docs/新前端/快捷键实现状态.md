## 焦点保护机制

快捷键系统已实现焦点检测机制：

1. **绝对拦截**（即使在输入框中也生效）：
   - `Ctrl + S` (保存)
   - `Ctrl + Z` (撤销)
   - `Ctrl + Shift + Z` (重做)
   - `Ctrl + Y` (重做)

2. **输入框保护**（在输入框中不触发）：
   - 所有其他快捷键在用户输入文本时不会触发

## 冲突处理

所有快捷键都已使用 `e.preventDefault()` 阻止浏览器默认行为，避免与浏览器快捷键冲突。
