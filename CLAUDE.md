# 该项目特别要求 - 非常重要！！！
为了保证最后能一键切换到嵌入式环境，你在写代码时，必须严格杜绝以下行为：
1. 杜绝硬编码绝对路径
不要写类似： C:\Users\Administrator\VideoSRT\models 或 D:\Anaconda\envs\srt\python.exe 要写： 基于当前脚本位置的相对路径。
2. 杜绝依赖系统环境变量 (FFmpeg)
在代码中优先检测项目根目录下是否存在 ffmpeg/bin/ffmpeg.exe。如果存在，强制使用该完整路径调用；如果不存在，再尝试系统命令 ffmpeg
3. 强制接管模型下载路径
所有模型、数据集等均应该储存在项目目录的指定位置，HuggingFace 和 Torch 默认会把模型下到 C 盘用户目录，必须纠正这种情况
